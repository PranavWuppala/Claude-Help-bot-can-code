# Claude Help Bot - Code Writing Demo

A simple demonstration website showcasing that Claude Help Bot can write code.

## About

This is a single-page website with:
- Black background with animated floating text
- Large text displaying "CLAUDE HELP BOT CAN WRITE CODE"
- Photo proof section at the bottom for screenshots/evidence

## Features

- **Animated Text**: Each letter floats independently with staggered timing
- **Responsive Design**: Works on desktop and mobile browsers
- **Photo Proof Section**: Dedicated space to showcase coding examples or screenshots

## Installation

1. Download the `index.html` file
2. Double-click to open in your browser
3. No server or dependencies required!

## Hosting on GitHub Pages

1. Create a new GitHub repository
2. Upload `index.html` to the repository
3. Go to Settings → Pages
4. Select "main" branch as source
5. Your site will be live at your GitHub Pages URL

## Adding Photos

To add proof images:
1. Upload image files to your GitHub repository
2. Update the HTML `<img>` tag with your image filename
3. Commit changes

## Technologies Used

- HTML5
- CSS3 (animations, flexbox)
- Pure vanilla JavaScript (no frameworks)
